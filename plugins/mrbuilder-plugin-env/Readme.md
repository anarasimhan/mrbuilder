mrbuilder-plugin-env
===
This plugin is designed to be used with (mrbuilder)[https://github.com/jspears/mrbuilder].

## Installation
```sh
  $ yarn add "mrbuilder-plugin-env" -D
```
## Configuration
In package.json
```json
{
 "name":"your_component"
 ...
 "mrbuilder":{
    "plugins":[
      "mrbuilder-plugin-env"
    ]

 }
}
```
